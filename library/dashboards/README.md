# Dashboard Library

Reusable dashboard definitions organized by category.

## Categories

- [Version Overview](./version-overview/) - Version status dashboards
- [Execution Monitoring](./execution-monitoring/) - Execution tracking dashboards
- [Project Health](./project-health/) - Project metrics dashboards

## Usage

```bash
# List all dashboards
kad library dashboards list

# Show dashboard details
kad library dashboards show version-overview/version-status

# Load dashboard
kad library dashboards load version-overview/version-status
```
